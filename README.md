# 🚀 Cosmic Space Shooter

A modern **HTML5 Canvas** arcade space shooter built with JavaScript featuring cinematic boss battles, Overdrive mode, Shield power-ups, dynamic difficulty, and a polished neon sci-fi interface.

---

## 🎮 Features

* 🚀 Smooth spaceship controls
* 👾 Progressive enemy waves
* 👑 Cinematic Boss Battles
* ⚡ Overdrive Mode
* 🛡 Shield Power-up
* 🔥 Rapid Fire Power-up
* 🎯 Three Difficulty Modes (Easy, Normal, Hard)
* ⏸ Pause Menu
* 🌌 Animated Space Background
* ✨ Particle Effects & Explosions
* 🏆 Highest Score System
* 🎖 Player Rank System
* 💀 Game Over Screen with Restart & Main Menu

---

## 🕹 Controls

| Key           | Action             |
| ------------- | ------------------ |
| ⬅➡ Arrow Keys | Move               |
| Space         | Shoot              |
| E             | Activate Overdrive |
| ESC           | Pause / Resume     |
| R             | Restart            |
